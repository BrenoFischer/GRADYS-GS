README para contador simples (rodando no servidor)

- Clonando repositório:
1. Clonar repositório
2. Criar virtual env
3. Ativar venv
4. Rodar pip install -r requirements.txt


- Acessando variaveis secretas:
1. Criar arquivo .env na pasta config
2. Colocar as variáveis no arquivo


- Rodando servidor para múltiplos clientes:
1. Com Redis instalado, rodar redis-server


-Rodando servidor conexão arduino:
1. Rodar python manage.py runserver


- Acessando contador:
1. Acessar localhost:8000
